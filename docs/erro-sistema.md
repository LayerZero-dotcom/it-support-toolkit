
#  Erros no Sistema – Diagnóstico e Solução

##  Sintomas
- Programas fechando sozinho
- Mensagens de erro no sistema
- Travamentos aleatórios
- Sistema instável

---

##  Possíveis causas
- Arquivos do sistema corrompidos
- Atualizações incompletas
- Falhas no sistema operacional

---

##  Diagnóstico

### Windows:
- Verificar erros no sistema
- Rodar verificação de integridade

### Linux:
```bash
dmesg | less
journalctl -xe
