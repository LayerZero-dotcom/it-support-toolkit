
#  Sem Internet – Diagnóstico e Solução

##  Sintomas
- Sem acesso à internet
- Wi-Fi conectado mas sem navegação
- Erro de rede

---

##  Possíveis causas
- Problema no roteador
- Falha no DHCP
- Configuração de rede incorreta
- DNS com erro

---

##  Diagnóstico rápido

### Windows:
```bash
ipconfig
ping 8.8.8.8
